# Módulo 8 — Prompts, Skills, MCP

**4 aulas · ~20min · Status: 📝 outline-only**

> Material a expandir.

---

## Aula 8.1 — Bom prompt (5 min)

**Tema:** Os 5 elementos de prompt sólido: contexto + formato + restrições + exemplos + critério de aceite. Anti-padrões comuns.

**🏆 Troféu:** Você reescreveu 1 prompt seu seguindo os 5 elementos e o resultado melhorou.

---

## Aula 8.2 — Skills (5 min)

**Tema:** SKILL.md — formato de instrução reutilizável. Criar primeira skill (ex: `gerar-conteudo`). Como o Claude descobre e invoca.

**Material relacionado:** `kit-template/skills/` (skills do kit)

**🏆 Troféu:** Sua primeira skill funciona — você digita `/<nome>` no Claude e ela executa.

---

## Aula 8.3 — MCP (5 min)

**Tema:** Model Context Protocol — conecta Claude a serviços (Gmail, Calendar, Notion, custom). Quando vale criar MCP próprio.

**Material relacionado:** `biblioteca/tecnico/mcp-arquitetura.md`

**🏆 Troféu:** Você ativou 1 MCP relevante pro seu caso (ex: Calendar pro Claude marcar reunião).

---

## Aula 8.4 — Multi-model (5 min)

**Tema:** Claude (principal) + Codex (review) + Gemini (visão). Cada um pra sua tarefa. OpenRouter pra economia.

**🏆 Troféu:** Você usou ≥ 2 modelos em tarefas diferentes na mesma sessão.
