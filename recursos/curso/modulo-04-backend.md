# Módulo 4 — Backend (FastAPI + Python + LLM)

**8 aulas · ~40min · Status: 📝 outline-only**

> Material a expandir. Outline + troféu abaixo — Felipe escreve o script detalhado sob demanda durante a S2/S3.

---

## Aula 4.1 — Projeto do zero (5 min)

**Tema:** `npx create-next-app` + `CLAUDE.md` + Git init + GitHub. Setup de projeto novo em 5min.

**🏆 Troféu:** Você tem repo no GitHub com CLAUDE.md + commit inicial.

---

## Aula 4.2 — FastAPI: balcão da cozinha (5 min)

**Tema:** FastAPI = framework Python que recebe pedidos HTTP. `POST /generate` retornando hello. Hot reload.

**🏆 Troféu:** Sua API responde em `http://localhost:8000` quando você faz POST.

---

## Aula 4.3 — Docker: empacotando (5 min)

**Tema:** Dockerfile básico (Python 3.11 + requirements + cmd). Build. Por que Docker pra deploy.

**🏆 Troféu:** `docker build .` roda sem erro.

---

## Aula 4.4 — LLM escreve conteúdo (5 min)

**Tema:** Chamar API Anthropic/OpenAI dentro do FastAPI. Prompt estruturado (AIDA). Parse JSON da resposta.

**🏆 Troféu:** Você manda input → LLM retorna conteúdo estruturado.

---

## Aula 4.5 — Modelos YAML (4 min)

**Tema:** Templates de conteúdo como YAML. Por que YAML > código hard-coded. Estrutura reusável.

**🏆 Troféu:** Você tem 2 modelos YAML diferentes prontos.

---

## Aula 4.6 — Templates HTML (5 min)

**Tema:** HTML + Mustache (variáveis) + CSS variables. Como o conteúdo vira layout.

**🏆 Troféu:** Você renderiza HTML com dados diferentes do mesmo template.

---

## Aula 4.7 — Renderer: HTML → output (5 min)

**Tema:** Playwright (ou alternativa) renderiza HTML em PNG/PDF. Resolução, espera de fontes.

**🏆 Troféu:** Você tem o primeiro arquivo renderizado.

---

## Aula 4.8 — Supabase em 5 min (5 min)

**Tema:** Criar tabelas, Storage pra arquivos, RLS (Row Level Security). Quando usar SQL direto vs cliente.

**🏆 Troféu:** Você salva e recupera dados do Supabase.
