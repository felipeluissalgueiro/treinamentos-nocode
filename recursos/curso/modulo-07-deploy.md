## Aula 7.1 — Git: 5 comandos (4 min)

**Tema:** `git add` · `commit` · `push` · `pull` · `status`. `.gitignore`. Fluxo mínimo dia-a-dia.

**🏆 Troféu:** Você fez commit + push sem precisar consultar.

---

## Aula 7.2 — Vercel (5 min)

**Tema:** GitHub → Vercel automático. Env vars. Domínio custom. Preview deploys por branch.

**🏆 Troféu:** Frontend no ar com URL Vercel funcionando.

---

## Aula 7.3 — Railway (5 min)

**Tema:** Docker → Railway. Variables. Logs. Volumes pra dados persistentes.

**🏆 Troféu:** Backend no ar respondendo no Railway.

---

## Aula 7.4 — Teste end-to-end (4 min)

**Tema:** Roda o fluxo completo em produção. Onde geralmente quebra (CORS, env, timeout). Como debugar.

**🏆 Troféu:** Você gerou um output em produção, ponta a ponta, sem ajustar nada local.

---

## Aula 7.5 — Quando NÃO construir (4 min)

**Tema:** APIs prontas (Resend, Twilio, Stripe) > construir do zero. Como avaliar build vs buy. Custo de manutenção.

**🏆 Troféu:** Você listou 3 partes do seu MVP que vale a pena usar serviço pronto.
