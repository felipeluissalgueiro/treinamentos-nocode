## Glossário Dev & Design

**Arquivo:** `glossario-dev-design.md`

Dicionário com ~930 termos técnicos organizados em 9 domínios:

1. Fundamentos, Linguagens e Paradigmas
2. Frontend Web e Mobile
3. Backend, APIs e Autenticação
4. Bancos de Dados e Dados
5. DevOps, Infra, Cloud, Observabilidade e Segurança
6. Arquitetura, Padrões, Versionamento, Testes e Metodologias
7. Inteligência Artificial, LLMs e Machine Learning
8. Design — UX, Pesquisa e Produto
9. Design — UI, Visual, Design Systems, Branding e Ferramentas

### Como usar com IA

```
Você é meu assistente de estudo. Tenho o glossário em mãos.
Quando eu usar um termo técnico que não entendo, me explique
em 2-3 linhas usando a definição do glossário + um exemplo
do contexto de quem está construindo produto com IA.
```

Ou para buscar um termo específico:

```
No glossário abaixo, o que significa [termo]?
Como esse conceito aparece na prática quando estou
desenvolvendo com Claude Code?

[cole a seção relevante do glossário]
```
