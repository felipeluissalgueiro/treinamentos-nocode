# Live 01 — Como Montar seu Próprio Harness de IA com Claude Code

> "Harness é o nome técnico para o que a maioria das pessoas ainda está tentando descobrir como fazer: um sistema operacional de IA — com contexto, memória e automações — que trabalha por você de forma consistente."

**Duração:** 2 horas | **Data:** 04/06/2026

---

## O que você vai aprender

1. Como o Claude Code funciona por baixo do capô — e o que o diferencia de um chat ou IDE com IA
2. O que é um harness e quais são suas 3 camadas: CLAUDE.md, STATE.md, skills
3. Como criar um CLAUDE.md que dá personalidade, escopo e regras ao agente
4. Como estruturar squads — agentes especializados com contexto e memória próprios
5. Como orquestradores e workers dividem responsabilidades
6. O que são hooks e como automatizam comportamentos sem você pedir
7. Como spawnar subagentes e conectar serviços externos via MCP
8. Por que multi-model importa e como usar cada modelo no papel certo

---

## O que você vai ser capaz de fazer depois dessa live

- Instalar e configurar o Claude Code no próprio ambiente
- Criar um CLAUDE.md estruturado com contexto real
- Montar um squad simples com persona, escopo e memória
- Criar uma skill funcional e invocá-la no harness
- Ter um esqueleto funcional pronto pra expandir
